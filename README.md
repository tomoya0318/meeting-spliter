## 使い方
基本は`main.py`を実行するだけです

#### 欠席者がいた場合
以下のような形式で入力してください
```
python main.py absent "休んだ人の氏名" "休んだ人の氏名2"
```

#### 人を追加したい場合
以下のような形式で入力してください
```
python main.py add "学年" "氏名"
```
学年は，`src/models.py`内にて定義しており，現在`B2`,`B3`,`B4`,`M1`,`M2`を登録しています．
適宜修正してください

#### 人を削除したい場合
以下のような形式で入力してください．氏名は，登録した時のものでお願いします．
```
python main.py remove "氏名"
```
